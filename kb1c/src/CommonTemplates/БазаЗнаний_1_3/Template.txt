Версия 1.3
Изменения:
- добавлена возможность подписки на новости для категорий
- - в новостях отражаются следующие изменения
- - - добавление / изменение / удаление категорий
- - - добавление / изменение / удаление статей
- - - добавление / изменение / удаление комментариев
- - - добавление комментариев в ответ на собственные комментарии
- добавлена новостная лента на главную страницу
- изменения на главной странице базы знаний
- - к категориям добавлена пиктограмма наличия подписки
- - к категориям добавлена информация о количестве статей в иерархии
- изменена форма краткого представления статьи
- - добавлена информация о количестве комментариев
- - добавлена информация о средней оценке
- добавлена возможность переключения страниц
- - для новостей на главной странице выводится по 7 позиций
- - для новостей на странице ленты выводится по 20 позиций
- - для статей на странице по категории выводится по 20 позиций
- добавлена обработка перехода по ссылкам внутри страницы (ранее они игнорировались)
- добавлена возможность установки сворачивания вложенного раздела статьи
- - добавлена настройка на форме редактирования раздела
- при переходе по внутренним ссылкам на статьи с указанным разделом происходит автоматическое позиционирование на данном разделе
- исправлена ошибка разбора стилей таблицы, сами стили в странице вложены в блок вывода таблицы (html)
- изменена система хранения и редактирования гиперссылок. формирование адреса гиперссылки унифицировано
- добавлена возможность обновления комментариев без полного обновления всей страницы статьи
- значительные изменения в интерфейсе пользовательской части базы знаний
- при открытии формы объекта для справочника "Статьи базы знаний" происходит автоматическое открытие пользовательского представления страницы
- обновление данных статистики базы знаний переведено в фоновый режим

Версия 1.2
Изменения:
- обновление базы знаний возможно выполнять по механизму из БСП. Для этого добавлен общий модуль "ОбновлениеИнформационнойБазыБазаЗнаний". Подробнее см. документацию по БСП и пример из демо конфигурации
- изменена система хранения пользователей базы знаний. основной плюс системы - возможность в будущем расширять управление правами и доступом.
- изменена система хранения и редактирования гиперссылок. основной плюс - возможность расширения для хранения различного вида гиперссылок
- расширены возможности редактирования статьи.
- - добавлена возможность добавления текста комментария, который видно только в режиме редактирования *
- - добавлена возможность добавления разделительных линий *
- в имени картинок и таблиц добавлена возможность использования нижнего прочерка (_)
- добавлена возможность создания тегов (ключевых слов)
- сильно изменена таблица стилей, отлажено отображение для IE6+, Chrome

* подробнее см. справку из справочника "Статьи базы знаний"
