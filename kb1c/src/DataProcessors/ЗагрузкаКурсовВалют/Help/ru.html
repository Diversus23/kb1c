Предназначен для загрузки курсов валют с сайта РБК.

Нажмите "Добавить все валюты", удалите из списка валюты, курсы которых не требуется загружать, в полях "с" и "по" укажите интервал загружаемых курсов и затем нажмите "Загрузить".

Реквизиты:

  • Список валют
    		•

    Валюты, курсы которых необходимо загрузить из сети Интернет.
    Заполняется при помощи команд "Добавить все валюты" и "Подобрать".
    Открыть валюту можно по двойному щелчку мыши.

  • С
    		•  Начало периода загрузки курсов валют.
  • По
    		•  Конец периода загрузки курсов валют.

    Команды:

  • Загрузить
    		•  Загрузить курсы выбранных валют за выбранный период.
  • Добавить все валюты
    		•  Добавить в список валют все валюты, у которых установлен флажок "Загружается из сети Интернет".
  • Подобрать
    		•  Подобрать валюты в список.

     

    См. также: Валюты, Валюта, Общероссийский классификатор валют, Общая информация о формах.
