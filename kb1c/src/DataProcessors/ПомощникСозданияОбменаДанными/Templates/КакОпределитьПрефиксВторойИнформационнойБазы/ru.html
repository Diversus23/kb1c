Префикс второй информационной базы

Для получения префикса второй информационной базы следует выполнить запуск программы, перейти в форму синхронизации данных. Из формы синхронизации данных следует вызвать помощник настройки синхронизации. Внизу на первой странице помощника следует перейти по гиперссылке "Узнать префикс этой информационной базы".
Если во второй информационной базе префикс не задан, то следует указать любой желаемый префикс. Этот префикс будет установлен во второй базе при настройке синхронизации данных.
