Форма предназначена для ввода адреса по шаблону. При заполнении формы возможно использование адресного классификатора.

  • Поле "Страна" позволяет выбрать страну адреса. Редактирование адресов за пределами России производится в общем поле, без учета частей адреса
  • Поле "Индекс" предназначено для ввода почтового индекса, соответствующего данному адресу.Если адрес найден а вдресном классификаторе, то индекс может быть проставлен автоматически.
  • Поле "Город, населенный пункт" предназначено для ввода региона, города и т.п. При необходимости можно выбрать или ввести части адреса детально, нажав на кнопку выбора.
  • Поле "Улица" предназначено для выбора или ввода улицы адреса.
  • Поля "Дом", "Корпус", "Квартира" позволяют ввести соответствующие части адреса. Можно изменить эти варианты, выбрав значения из выпадающего списка или добавить дополнительные поля для ввода других предопределенных частей адреса, таких как "Строение", "Литера", "Сооружение", "Участок", "Офис", "Бокс", "Помещение", "Комната". Для этого необходимо воспользоваться кнопкой "Добавить".

Например, для ввода адреса "Санкт-Петербург, Пискаревский проспект, дом 2, корпус 2, литер Щ, офис 12345" необходимо ввести:

  1. В поле "Город, населенный пункт" - Санкт-Петербург г.
  2. В поле "Улица" - Пискаревский пр-кт.
  3. В поле значения "Дом" указать 2.
  4. В поле значения "Корпус" - 2.
  5. Для ввода литеры необходимо добавить поле кнопкой "Добавить", выбрать из выпадающего списка значение "Литера" и проставить в поле значение Щ.
  6. Вместо "Квартира" выбрать из выпадающего списка значение "Офис" и ввести в поле значения 12345.

Аналогично, для ввода адреса, содержащего другие значения полей, например "Москва, улица Адмирала Лазарева, домовладение 89, корпус 16, строение 1 необходимо:

  1. В поле "Город, населенный пункт" ввести Москва г.
  2. В поле "Улица" - Адмирала Лазарева ул.
  3. Вместо "Дом" выбрать из выпадающего списка значение "Домовладение", ввести в поле значения 89.
  4. В поле значения "Корпус" указать 16.
  5. Добавить поле кнопкой "Добавить", выбрать из выпадающего списка значение "Строение", проставить в поле значение 1.
  • Поле "Комментарий" позволяет указать произвольный комментарий к адресу, который не будет включаться в его представление и выводится на печать.
  • Кнопка "Проверить заполнение" анализирует корректность заполнения адреса и выводит список ошибок.

Из меню "Все действия" также доступны команды "Очистить адрес" и "Загрузить классификатор"
