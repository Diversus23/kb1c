В форме выполняются персональные настройки электронной подписи (ЭП).

Настройка Описание
Действия при сохранении с ЭП

Задает поведение при выполнении команды "Сохранить с ЭП":

  • Сохранять все подписи - все подписи сохраняются;
  • Спрашивать - выбор сохраняемых подписей осуществляется в форме.
Расширение для зашифрованных файлов Задает расширение для зашифрованных файлов при отправке по почте, при выполнении команды "Сохранить как" и при экспорте папки.
По умолчанию "p7m".
Путь модуля криптографии (для веб клиента под управлением Linux) Задает путь модуля криптографии для операций с файлами (расшифровка, подписание, проверка подписи, просмотр сертификата).
Личный сертификат для шифрования

Персональный сертификат электронной подписи (ЭП) для шифрования данных. При выборе сертификатов для шифрования данный сертификат будет всегда отмечен. Это нужно для того, чтобы нельзя было зашифровать данные без возможности их расшифровки.
