Как настроить подпись к письму?


Настройка подписей производится на одноименной закладке. Можно создать подпись двух видов: для новых сообщений и для сообщений при ответе или пересылке.
Установите флаг Включить для нужного вида подписи. Укажите формат подписи. Создание подписи в формате HTML позволит форматировать текст и вставлять в него картинки.

Важно. Создание подписей в формате HTML доступно, если это предусмотрено настройками программы. Данную настройку можно изменить в разделе Администрирование/Органайзер отредактировав значение флага Исходящие письма в формате HTML.

Важно. Если для учетной записи настроена подпись, то при создании письма подставляется подпись учетной записи.

Как указать, что для всех отправляемых писем по умолчанию необходимо уведомление о доставке и прочтении?


Данные настройки можно выполнить на закладке Отслеживание писем. Установите флаги Уведомление о прочтении и Уведомление о доставке. Эти значения будут использоваться для каждого исходящего письма как значения по умолчанию, при этом их можно будет изменить.

Как настроить порядок ответов на запросы уведомлений о прочтении?


Данные настройки можно выполнить на закладке Отслеживание писем. Доступны следующие варианты:

  • Всегда отправлять уведомление - после установки флага Рассмотрено в письме будет отправлено уведомление о прочтении.
  • Никогда не отправлять уведомление - запросы на отправку уведомления о прочтении будут игнорироваться.
  • Запрашивать перед тем как отправить уведомление - при установке флага Рассмотрено в письме будет задан вопрос о отправке уведомления о прочтении.


Как указать что в форме исходящего письма при ответе или пересылке необходимо отображать тело письма основания?


Установите флаг Отображать тело исходного письма при ответе или пересылке на закладке Прочее. Это значение будут использоваться для каждого исходящего письма как значение по умолчанию, при этом в самом письме его можно будет изменить.


Как указать что в тело исходящего письма при ответе или пересылке необходимо включать тело письма основания?


Установите флаг Включать тело исходного письма при ответе или пересылке на закладке Прочее. Это значение будут использоваться для каждого исходящего письма как значение по умолчанию, при этом в самом письме его можно будет изменить.
