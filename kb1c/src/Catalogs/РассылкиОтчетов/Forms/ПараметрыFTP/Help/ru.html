Позволяет указать параметры, необходимые для подключения к FTP ресурсу.
Поле "Сервер и каталог" является обязательным для заполнения, остальные поля не обязательны.

Реквизиты:

  • Сервер и каталог
    		•  Путь к FTP ресурсу в формате "ftp://сервер/каталог/".
  • Порт
    		•  Порт FTP ресурса на сервере (по умолчанию 21).
  • Логин
    		•  Имя пользователя для доступа к FTP.
  • Пароль
    		•  Пароль пользователя для доступа к FTP.
  • Пассивное соединение
    		•  Флажок использования пассивного соединения при подключении к серверу.

    Команды:

  • Заполнить
    		•  Позволят ввести все параметры одной строкой.
