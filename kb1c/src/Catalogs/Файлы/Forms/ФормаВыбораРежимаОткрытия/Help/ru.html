Для открытия файла используйте:

  • клавишу Enter 
  • двойной щелчок левой кнопкой мыши
  • команду "Открыть файл"

 

Варианты открытия файла:

Только для просмотра Файл не будет помечен как занятый Вами для редактирования и откроется внешним приложением в режиме "только просмотр"
Для редактирования Файл будет помечен как занятый Вами для редактирования и откроется внешним приложением с возможностью внесения и сохранения изменений

См. также: Общая информация о файлах.
