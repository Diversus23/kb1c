Правило обработки электронной почты

Справочник Правила обработки электронной почты предназначен для создания и хранения условий автоматического распределения входящей и исходящей корреспонденции по папкам. Правила настраиваются для Учетной записи электронной почты.

Как настроить правило?

Как настроить правило?
  • Условие - текстовое описание условия, по которому будут отбираться письма. Текст может быть сформирован автоматически, исходя из настроенных условий.
  • Помещать в папку папка, в которую будут помещены письма, которые соответствуют условию.
  • Условия - указываются поля, по которым будет происходить отбор, значения полей отбора, при необходимости группируются поля отбора.
