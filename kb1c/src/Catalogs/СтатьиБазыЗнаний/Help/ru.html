База знаний работает по принципу Википедии. Любой пользователь программы (имеющий доступ) может свободно отредактировать любую статью, и изменения сразу же учтены и отображены в базе знаний.

Основы

Редактировать статью в базе знаний просто. При просмотре статьи следуйте ссылке "Редактировать" . Откроется страница для редактирования, на ней есть поле форматированного документа, заполненное исходным текстом статьи, который содержит особые символы и конструкции (разметку), описанную ниже.

Редактирование

Затем:

  • Внесите желаемые изменения в исходный текст;
  • Кратко опишите смысл своей правки в строке "Описание изменений";
  • Для проверки правильного отображения страницы, нажмите кнопку "Просмотр";
  • Если со страницей все в порядке, опубликуйте свою новую версию нажатием кнопки "Записать".

Пожалуйста, пришите с нейтральной точки зрения, избегайте предвазятости и излишней эмоциональности; указывайте источники информации - это поможет другим проверить и дополнить вашу работу. В общем следуйте правилам, это не трудно и полезно.

Дополнительные возможности

  • Выше текстовой формы редактирования находятся кнопки для форматирования текста, ниже - панель быстрой вставки специальных символов, элементов разметки и шаблонов. Среди них кнопка обработки текста.
  • Кнопка "Показать отличия" позволяет сравнить исходный код с тем, что у нас в окне редактирования.

Также доступны следующие возможности:

  • Признак "Малое изменение" позволяющий отметить ваши изменения как незначительные;
  • Признак "Включить эту страницу в список наблюдения" для наблюдения за дальнейшими изменениями этой статьи.

В будущем планируется возможность открытия для редактирования в текстовых редакторах. Следите за изменениями.

Кроме того, можно обсуждать статью с другими участниками. Практически у каждой статьи существует параллельная страница с ее обсуждением.

Специальная разметка

Ниже приведена правочная информация по редактированию статей. Прежде чем применять эти "инструменты" на "живом теле" статьи, поэксперементируйте, пожалуйста в "Песочнице" (или используйте кнопку "Просмотр").

Ссылки, URL

Вставка гиперссылок выполняется при помощи нажатия на кнопку или при помощи горячей клавиши Alt+H. При этом выделенный текст подставляется в виде представления ссылки. Гиперссылки могут быть трех видов: внутренняя (на статью базы знаний), внешняя (на интернет-ресурс), эл. почту (для быстрого создания письма).

Форматирование текста

Для указания в тексте комментариев, можно применять конструкцию:

<!-- Текст комментария -->

Данный текст не отображается на странице и виден только в режиме редактирования. Можно использовать для важных пометок.

Оглавление статьи

Если в статье содержится, по меньшей мере, четыре раздела, то панель "Содержание" (с оглавлением статьи) автоматически добавляется на страницу, непосредственно перед первым заголовком (т.е. сразу после вступительного раздела). Положение заголовка регулируется значением реквизита "Вывод заголовка" на странице "Содержание".

Горизонтальная линия

Для подстановки горизонтальной разделительной линии используется строка состоящая из четырех тире (----) расположенная на новой строке. Линия используется только для оформления текста и не делит форму на разделы.

ВНИМАНИЕ: В данной строке не может быть расположено никаких других символов (в т.ч. точек, тире, пробелов) кроме четырех тире, в противном случае разделитель не будет распознан.

Картинки

Вставка картинок выполняется при помощи нажатия на кнопку  или при помощи горячей клавиши Ctrl+P. Если выделенный текст (или позиция курсора) является текстом указания картинки, откроется форма с выбранной картинкой, иначе будет предложено добавить новую картинку. Картинки на форме могут повторяться, для этого необходимо продублировать размеченный текст с картинкой.

Размеченный текст выглядит следующим образом: [picture='Имя картинки']

Таблицы

Вставка таблиц выполняется при помощи нажатия на кнопку или при помощи горячей клавиши Ctrl+T. Если выделенный текст (или позиция курсора) является текстом указания таблицы, откроется форма с выбранной таблицей, иначе будет предложено добавить новую таблицу. Таблицы на форме могут повторяться, для этого необходимо продублировать размеченный текст с таблицей.

Размеченный текст выглядит следующим образом: [table='Имя картинки']

Сноски

Вставка сноски выполняется при помощи нажатия на кнопку  или при помощи горячей клавиши Ctrl+R. Если выделенный текст (или позиция курсора) является текстом указания сноски, откроется форма редактирования сноски, иначе будет предложено добавить новую сноску. Сноски на форме могут повторяться.

Размеченный текст выглядит следующий образом: #n. Где n - номер сноски.

Повторное использование сноски

При необходимости повторного использования сноски необходимо скопировать размеченный текст и подставить в необходимое место.

Группировка сносок

При необходимости группировки сносок по разным именам (разделам), сноске должна быть указан группа, к которой она относится. Имя группы указывается в форме редактирования сноски.
