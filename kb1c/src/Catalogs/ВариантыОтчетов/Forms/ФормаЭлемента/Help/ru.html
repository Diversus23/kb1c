Справочник предназначен для хранения вариантов отчетов и настройки их размещения в командном интерфейсе.

Реквизиты:

  • Наименование
    		•  Представление варианта отчета в списках.
  • Описание
    		•  Подробная информация о варианте отчета, выводится как подсказка в панелях отчетов.
  • Только для автора
    		•  Доступность варианта отчета только автору или всем пользователям данного отчета.
  • Автор
    		•  Автор варианта отчета.
  • По умолчанию виден в панелях отчетов всех пользователей
    		•  Значение по умолчанию флажка "Видимость" для всех пользователей в панелях отчетов.

    Таблица "Размещение":

  • Использование
    		•  Если выключен, то отчет не будет размещаться в группе раздела.
  • Раздел, Группа
    		•

    Подсистема, в которой необходимо разместить вариант отчета.
    Подсистемы верхнего уровня - разделы, остальные - группы.
    При размещении в разделе вариант отчета будет выводиться без группы.

  • Важность
    		•  Способ выделения варианта отчета в группе:
    Важный – вариант отчета будет выделен жирным шрифтом.
    См. также – вариант отчета будет выведен в отдельной группе внизу панели отчетов.

    Команды:

  • Просмотр
    		•  Открыть отчет для формирования и просмотра.
  • Сбросить настройки пользователей
    		•  Сбросить пользовательские настройки в панелях отчетов по данному варианту отчета.
