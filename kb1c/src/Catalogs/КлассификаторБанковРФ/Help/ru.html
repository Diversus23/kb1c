В классификаторе банков РФ хранятся сведения о банках: БИК, корреспондентский счет, данные о местонахождении, телефоны для связи с сотрудниками, почтовый адрес банка. Эти сведения используются при заполнении реквизитов банковских счетов и в других объектах программы.

По команде Загрузить классификатор можно обновить сведения о банках с сайта РБК или из поставляемого на диске 1С:ИТС "Общероссийского классификатора банков" (ОКБ).

Также можно настроить обновление классификатора с сайта РБК автоматически по расписанию с помощью регламентного задания Загрузка классификатора банков РФ с сайта РБК.
